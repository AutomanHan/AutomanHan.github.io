Shell脚本--处理文件

1、文件和目录列表

1.1基本列表功能

ls
ls -F 区分文件和目录

ls -a 显示包括隐藏文件在内的所有文件

ls -FR 递归选项

ls -l –time=atime test_one 显示出已经更改过的文件访问时间。
ls -i 查看文件或目录的incode编号。

1.2显示长列表

ls -l 产生长列表格式的输出，包含了目录中每个文件的更多相关信息

  • 文件类型，比如目录（d）、文件（-）、字符型文件（c）或块设备（b）；
  • 文件的权限；
  • 文件的硬链接总数；
  • 文件属主的用户名；
  • 文件属组的组名；
  • 文件的大小（以字节为单位）；
  • 文件上次修改时间；
  • 文件名或目录名

    1.3过滤输出列表

    ls命令能够识别标准通配符，并在过滤器中用它们进行模式匹配：
  • 问号（？）代表一个字符；
  • 型号（*）代表零个或多个字符。

在过滤器中使用星号和问号被称为文件扩展匹配符

1.4 处理文件

1.4.1 创建文件

touch 创建文件
改变文件的修改时间，同时并不需要改变文件的内容

1.4.2 复制文件

ln code_file hl_code_file

ln -s code_file sl_code_file

rm -i fall

-i参数提示是否真的要删除该文件。
### 1.5 处理目录
**mkdir** 创建目录  
**rmdir** 删除空目录（只能删除空目录）  
**rm -r** 递归删除目录及文件
### 1.6 查看文件内容
**file** 查看文件类型    
**cat** 显示整个文件， -n参数会加上行号，-T可以不让制表符出现，但会用^I字符组合替换
文中的所有制表符。  
**more**  分页工具，可以通过空格键或回车键以逐行向前的方式浏览
文本文件，浏览完成之后，按q键退出；  
**less**  more的升级；  
**查看部分文件**  
**tail**  显示文件最后几行的内容，-n参数修改所显示的行数，
-f允许在其他进程使用该文件时查看文件的内容，这是实时监测系统日志的绝妙方式。  
**head** 显示文件开头那些行。
